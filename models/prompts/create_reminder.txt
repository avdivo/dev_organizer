1. Создай список напоминаний. Соблюдай правила:
   - Проверь даты. если есть ошибки в напоминании (например указанная дата прошла):
     запиши в "APScheduler": None, а в "answer": отказ, причину отказа с текстом напоминания, напоминание не ставится.
   - если говорится о нескольких напоминаниях, создай отдельное напоминание для каждой
   - сохраняй смысл напоминания в каждом
   - добавляй в каждое напоминание дату (если она есть)
   - если напоминание периодическое добавь период напоминания
2.  Для цифр подбери единицы измерения в полк  "numbers".
   Если не указаны - подбери по контексту.
   Преобразуй единицу измерения в каноническую форму —
   в именительном падеже, в единственном числе, без сокращений и спецсимволов.
   Правила:
   - по умолчанию для денег применяй рубли
   - номер года - год, номер дня - день, номер квартала - квартал
   - люди, сотрудники, работники - человек
   - животные, предметы, действия, пачки, упаковки - штуки
   - полка, место, ячейка, бокс, здание... порядковый номер - номер
   - баллы, рейтинг, оценка - бал

 4. Для напоминаний:
- для словаря APScheduler используй формат даты ISO 8601
- если время не указано ставь 00:00:00
- вычисляй дату и время первого напоминания в будущем. Если время сработки прошло? СТАВЬ СЛЕДУЮЩЕЕ!
- для понятий завтра. в понедельник, через месяц и др. вычисляй дату и записывай в
  "datetime_reminder" (поле обязательное)
- для периодов следующий месяц, следующую неделю и др. ставь дату начала периода

Примеры напоминаний для APScheduler:
{
  "trigger": "date",
  "run_date": "2025-06-17T18:00:00+03:00",     // Дата и время запуска (обязательное)
}

{
  "trigger": "interval",
  "weeks": 0,                                    // Интервал в неделях
  "days": 0,                                     // Интервал в днях
  "hours": 0,                                    // Интервал в часах
  "minutes": 30,                                 // Интервал в минутах
  "seconds": 0,                                  // Интервал в секундах
  "start_date": "2025-06-17T09:00:00+03:00",     // Дата начала (необязательное)
  "end_date": "2025-07-01T09:00:00+03:00",       // Дата окончания (необязательное)
}

{
  "trigger": "cron",
  "year": "2025",                                // Год
  "month": "1",                                  // Месяц (1–12)
  "day": "1",                                    // День месяца (1–31)
  "week": "1",                                   // Номер недели (1–53)
  "day_of_week": "mon",                          // День недели (mon–sun или 0–6)
  "hour": "0",                                   // Час (0–23)
  "minute": "0",                                 // Минута (0–59)
  "second": "0",                                 // Секунда (0–59)
  "start_date": "2025-01-01T00:00:00+03:00",     // Дата начала (необязательное)
  "end_date": "2025-12-31T23:59:59+03:00",       // Дата окончания (необязательное)
}
