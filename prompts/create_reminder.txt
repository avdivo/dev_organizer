Не пиши когда будешь напоминать, только содержательную часть сообщения..

        "metadata": {
            "text": напоминание 1 (обязательное поле)
            "price": сумма/цена,
            "datetime_create": создания заметки (обязательное поле)
            "datetime_reminder": когда напомнить первый раз время сработки прошло? СТАВЬ СЛЕДУЮЩЕЕ!
            "quantity": количество
            "number": номер или цифра которую нельзя отнести к имеющимся (например телефон)
            "rating": рейтинг
        },
        "APScheduler": {

        },
        "answer": короткий отчет об успешном создании. Сообщение и дата и время первого напоминания.
    },
    {
        "metadata": {
            "text": напоминание 2,
             ...
        },
        "APScheduler": {

        }
    }
    ...

4. Для напоминаний:
- для словаря APScheduler используй формат даты ISO 8601
- если время не указано ставь 00:00:00
- вычисляй дату и время первого напоминания в будущем. Если время сработки прошло? СТАВЬ СЛЕДУЮЩЕЕ!
- для понятий завтра. в понедельник, через месяц и др. вычисляй дату и записывай в
"datetime_reminder" (поле обязательное)
- для периодов следующий месяц, следующую неделю и др. ставь дату начала периода
